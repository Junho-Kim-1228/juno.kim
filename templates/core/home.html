김준호

정적 실험 페이지는 끝냈고, 이제는 EC2 위에서 돌아가는 Django 사이트로 정리했습니다. Nginx가 요청을 받고, Gunicorn이 Django 애플리케이션을 서비스하는 구조를 전제로 만든 시작점입니다.

소개

웹, AI, 자동화 쪽 작업을 합니다. 지금은 실제 서비스 구조를 올리는 단계입니다.

스택

  • Browser → juno.kim → DNS
  • EC2 → Nginx → Gunicorn
  • Django → SQLite by default
  • Production DB는 `DATABASE_URL`로 교체 가능
